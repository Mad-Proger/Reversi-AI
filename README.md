# Reversi-AI
AI to play Reversi (on 8 by 8 field) using neural network to evaluate board position

## Visualizer
GUI to play with AI using SFML.

External dependencies:
* sfml-graphics.lib
* sfml-window.lib
* sfml-system.lib

Differences may apper in case of static or debug build

You also have to add SFML include directory into search path
